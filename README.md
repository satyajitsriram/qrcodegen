# QR Code Generator 🧾

A fast, minimal QR code generator built with **React**, **Vite**, and **Tailwind CSS**. 

🔗 Enter any valid URL  
🎯 Generate a scannable QR code  
🎨 Customize its color  
⬇️ Download it instantly as a PNG  
🌗 Switch between dark/light mode  


---

### ⚙️ Tech Stack
- React + Vite
- Tailwind CSS
- `qrcode.react`
- Deployed on Vercel
